Java Shell / Console App
Учебный проект — консольное приложение на Java, написанное в процессе изучения языка.
О проекте
Это своего рода "оболочка" (shell) с набором команд, которая развивалась по мере изучения Java. Проект не преследует коммерческих целей — это просто практика и эксперименты с кодом.
Что реализовано

Система команд с обработкой ввода
Калькулятор
Блокнот (сохранение текста)
Дата и время
Простая система магазина с балансом и покупками
Централизованная обработка ошибок

Стек

Java
Стандартная библиотека (Scanner, ArrayList, LocalDate/LocalTime)

Статус
В разработке. Проект продолжает развиваться по мере изучения новых концепций Java.
